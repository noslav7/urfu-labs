# Лабораторная работа 10 - пункт 2.2 (вариант Animals)

Папка с решением: `src/main/java/ru/urfu/labs/lab10/point2_2`  
Файлы данных: `src/main/resources/lab10/point2_2`

## Кодировка (Windows PowerShell)

Перед запуском Java-примера выполните:

```powershell
chcp 65001
```

Это переключит консоль в UTF-8 и устранит вывод вида `?????` вместо русского текста.

## Что реализовано

- `2.2.1` Создан собственный XML-файл `animals.xml` по варианту "Домашние животные".
- `2.2.2` Добавление нового животного в XML.
- `2.2.3` Поиск животных по имени или по виду.
- `2.2.4` Удаление животных по имени.

## Как это реализовано в коде

- Запись XML выполняется через `Transformer` (`TransformerFactory.newInstance().newTransformer()`).
- Поиск реализован через Java Stream API (`filter()` по `name` и `species`).
- Удаление реализовано через DOM API: у найденного элемента берется `parentNode` и вызывается `removeChild(...)`.

## Почему поиск изменен

В исходной формулировке был пример для книг (поиск по автору/году).  
Для варианта **Animals** логично искать по полям животных, поэтому здесь реализован поиск:

- по имени животного;
- по виду (например, кот, собака, попугай).

## Меню программы

- `1` - показать всех животных
- `2` - добавить животное
- `3` - найти по имени
- `4` - найти по виду
- `5` - удалить по имени
- `0` - выход

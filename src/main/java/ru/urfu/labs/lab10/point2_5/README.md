# Лабораторная работа 10 - пункт 2.5 (Excel парсер)

Папка с решением: `src/main/java/ru/urfu/labs/lab10/point2_5`

## Что доработано

- Добавлены **подробные сообщения об ошибках** при чтении Excel.
- Для каждой ошибки выводится **рекомендация**, как ее исправить.
- Реализована возможность **повторного запуска чтения** после исправления ошибки без перезапуска программы.

## Какие ошибки обрабатываются

- файл не найден;
- путь указывает не на файл;
- неверное расширение (не `.xlsx`);
- отсутствует запрашиваемый лист (с выводом доступных листов);
- проблемы чтения (доступ/блокировка файла);
- поврежденный или неверный формат Excel.

## Запуск интерактивного режима

```powershell
mvn -q "-Dexec.mainClass=ru.urfu.labs.lab10.point2_5.Point2_5Runner" org.codehaus.mojo:exec-maven-plugin:3.5.0:java
```
